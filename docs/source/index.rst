Welcome to the Cloud CIX user guide!
====================================

This website is a user manual for beginners getting setup with the Cloud CIX platform, taking you through creating a virtual machine and installing basic unix tools to begin planning your own bioinformatics workflows. This website does not offer assistance in setting up an account, which is handled by the admin team at Cloud CIX.

.. note::

   If any of the URLs or code blocks no longer work please open an issue at `https://github.com/BarryDigby/Cloud_CIX/issues <https://github.com/BarryDigby/Cloud_CIX/issues>`_ describing both the error and location of the error. Please refrain from posting technical questions to the issue tracker.
